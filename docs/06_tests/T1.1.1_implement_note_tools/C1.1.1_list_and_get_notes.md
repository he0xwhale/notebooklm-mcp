# Test Case C1.1.1: List and Get Notes Verification

## 1. Objectives

Verify that the `note_list` and `note_get` tools function correctly on an authenticated NotebookLM session.

## 2. Prerequisites

- Authenticated session cookies in `/home/xor/.local/share/notebooklm-mcp/chrome_profile`.
- At least one note exists inside the notebook.

## 3. Test Steps

1. Initialize the browser using Playwright with the user profile.
2. Navigate to the notebook (uses active notebook from library or CLI arguments).
3. Switch to the Studio tab.
4. Execute `listNotes()`:
   - Verify that scrolling triggers successfully.
   - Verify that custom custom Web Components `artifact-library-note` are located.
   - Assert that the returned note count is greater than 0.
5. Select a note from the retrieved list.
6. Execute `getNoteContent(noteTitle)` using the title of the note:
   - Verify that the card is clicked.
   - Verify that the text content of the note is successfully extracted.
   - Verify that the Esc key closes the note editor dialog.
7. Return results.
